# Rust Unicode Wave Function Collapse

Updated version of [this repo](https://github.com/JeyyGit/unicode-wave-function-collapse) written in rust.
## usage
```bash
$ cargo build --release
```
```bash
$ .\target\release\uwfc.exe --help
Create pattern with wave function collapse.

Usage: uwfc.exe [OPTIONS]

Options:
      --height <VALUE>  height of generated pattern [default: 10]
      --width <VALUE>   width of generated pattern [default: 10]
      --n-iter <VALUE>  how many patterns generates [default: 1]
  -h, --help            Print help
  -V, --version         Print version
```
